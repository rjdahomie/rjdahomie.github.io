+ DNA Sequence Similarity Checker
This C++ terminal based program finds similarity between biological sequences. The program compares nucleotide or protein sequences and calculates a score based on the similarity of the DNA strands using Dynamic Progreaming. This task would normally take hours based on the DNA strand length. However this program reduces that time to seconds, a 99% decrease.
